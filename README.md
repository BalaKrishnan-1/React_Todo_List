# Todo-List Using ReactJS CRA CreateReact App
<p>📌 React Todo List App

A simple and efficient Todo List application built using React. This project helps users manage daily tasks with an intuitive and responsive interface.

🚀 Features
✅ Add new tasks
🗑️ Delete tasks
✏️ Mark tasks as completed
🔄 Dynamic UI updates using React state
📱 Responsive design

# ⚙️ Installation & Setup <br>

### Clone the repository
```terminal
git clone https://github.com/BalaKrishnan-1/React_Todo_List.git
```

### Navigate to the project folder
```terminal
cd React_Todo_List
```

### Install dependencies
```terminal
npm install
```

### Start the development server
```terminal
npm start
```
